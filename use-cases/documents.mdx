title: Gestión de documentos
description: 'Descubre cómo trabajar con documentos en Plumaa ID'
---

El caso de uso por excelencia es el de firmado de documentos electrónicos. Esencialmente, se trata de un procedimiento que permite a una persona física o moral, a través de un dispositivo electrónico, **dar fe de su voluntad de aceptar un contenido digital**.

La creación de documentos es el principal evento en el que se basa el ecosistema de Plumaa ID. Los documentos son creados y conservados a través de certificados de conservación emitidos por un PSC y/o un certificado de conservación en la blockchain.

En Plumaa ID es posible firmar un documento a través de las siguientes opciones:

<Tabs>
  <Tab title="Vínculo entre usuario y organización">
    A través de este mecanismo, una organización establece un vínculo con un usuario. Una vez establecido, este servirá como permiso para que la organización invite al usuario a firmar documentos. Este vínculo puede ser removido por el usuario en cualquier momento.
    <Info>
      Las invitaciones a firmar enviadas previo a la desvinculación seguirán siendo válidas.
    </Info>
  </Tab>
  <Tab title="Código QR">
    A través de un código QR, un usuario puede firmar un documento sin necesidad de estar vinculado a una organización. Este mecanismo es ideal para firmar documentos de manera esporádica o para usuarios que no desean establecer un vínculo con una organización.

    <Info>
      Los miembros de la organización que creó el documento podrán ver el código QR de invitación, además de un link que podrán compartir para firmar.
    </Info>
  </Tab>
</Tabs>

## Crear un documento

Las organizaciones pueden crear documentos a través de la [aplicación web](https://app.plumaa.id).

<Note>Cada organiazción cuenta con 5 documentos con certificado NOM-151 de cortesía, y 25 documentos con certificación en blockchain.</Note>

<Tip>Te recomendamos nuestra [guía de creación de documentos](/advanced/create-signature-request)</Tip>
