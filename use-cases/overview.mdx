Plumaa ID busca ser la solución por excelencia para **firmado digital**. A raíz de esta especialización, múltiples casos de uso se derivan de la plataforma.

El objetivo de Plumaa ID es facilitar infraestructura que permita a cualquier tipo de usuario dotar de propiedades de **integridad, autenticidad y no repudio** a mensajes de datos sin limitación a documentos.

Esta particularidad de nuestra infraestructura permite alojar la creatividad de desarrolladores y empresas para crear soluciones innovadoras y seguras utilizando el marco legal sobre el cuál Plumaa ID se sustenta.
