En Plumaa ID, el inicio de sesión a la aplicación web es sencillo y completamente basado en la firma electrónica avanzada.

El mecanismo general se describe en la [sección de autenticación de casos de uso](/use-cases/authenticate#como-verifico-la-identidad-de-un-usuario). Sin embargo, para efectos prácticos, es suficiente con escanear el código QR que se presenta al inicio de la aplicación:

<img width="auto" src="/images/guides/webapp/login/login.png" />

## ¿Cómo inicio sesión?

Para iniciar sesión, sigue estos pasos:

1. Abre tu aplicación Plumaa ID
2. Escanea el código QR que se muestra en la pantalla de inicio
3. Firma el mensaje de autenticación que te muestra tu aplicación
4. ¡Listo! Ya puedes acceder a la aplicación web de Plumaa ID
