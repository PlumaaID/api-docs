Plumaa ID tiene dos principales componantes:

- Aplicación Móvil: Para firmar documentos y autenticarse en servicios digitales.
- Aplicación Web: Para gestionar documentos y configuraciones de cuenta.

Las guías de Plumaa ID están organizadas por secciones de acuerdo a la aplicación que se esté utilizando.
